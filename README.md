# Increment-Decrement-Counter-App

A simple Increment Decrement Counter App built using React Native and Expo.

## Features

- Increase Counter
- Decrease Counter
- Reset Counter
- Simple and Clean UI
- Beginner-friendly project

---

# Technologies Used

- React Native
- Expo
- JavaScript

---

# Requirements

Before running this project, install:

- Node.js
- npm
- Expo Go App (Android/iOS)

---

# Installation

## Step 1: Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Counter-App.git
```

---

## Step 2: Open the Project Folder

```bash
cd Counter-App
```

---

## Step 3: Install Dependencies

```bash
npm install
```

This command will automatically install all required packages.

---

## Step 4: Start the Expo Development Server

```bash
npm start
```

or

```bash
npx expo start
```

---

## Step 5: Run the App

You have three options:

### Android

Press

```
a
```

or run

```bash
npm run android
```

---

### Web

Press

```
w
```

or

```bash
npm run web
```

---

### Physical Mobile

1. Install **Expo Go** from Play Store.
2. Connect phone and PC to the same Wi-Fi.
3. Scan the QR Code shown in the terminal.

The app will open automatically.
